> Fire alert tool

### Intital setup

- Clone this repo

- run `npm i`

- run `npm run build`

- run `npm link`

### About

- For abstract [check here](https://github.com/knrt10/alert-tool/blob/master/abstract.docx) 

- For report [check here](https://github.com/knrt10/alert-tool/blob/master/report.docx)
